# bigg-swap-core
Core contracts for the BIGG AMM.

## Introduction to the BIGG AMN
To maintain stability, BIGG is equipped with it's own AMM that utilizes [Curve](https://curve.fi)'s formula, allowing for low-slippage BIGG swaps.
